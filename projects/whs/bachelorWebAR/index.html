Demos und Testanwendungen zur Bachelorarbeit

Hier findet man alle Demoanwendungen zur Beachelorarbeit "Webbasiertes Augmented Reality mit WebXR und JavaScript."
  • AR Fahrradkonfigurator
    Eine Augmented Reality Webanwendung zur Betrachtung von Fahrrädern. Dies ist nur eine Demo. Die Fahrradmodelle sind erfunden und gehören zu keiner realen Marke.
    •
  • Minimalistische WebXR Demo
    Eine kleine, minimalistische WebXR Demo mit einfachem SLAM Tracking. Ein Objekt wird einfach schwebend im Raum platziert und an dieser Position gehalten.
    •
  • Kamera Demo
    Eine Webanwendung, die auf die Systemkamera zugreifen kann. Ermöglicht wird dies durch JavaScript und die MediaDevices API.
    •
  • WebGL Shader Showroom
    Eine Demo, die die Unterschiede zwischen verschidenen Shadertypen zeigt. Zusätzliche Optimierungen für die WebGL Darstellung können ausprobiert werden.
    •
  • WebGL Shader Demo
    Eine Demo, die alle WebGL grundlegenden und physischen Shader und Materialien präsentiert.
    •
